# Learn System for OpenClaw

> 完整的学习管理系统，支持学习目标管理、课程学习、进度追踪、书签管理和知识缓存。

## ✨ 特性

- 🎯 **学习目标管理** - 创建、追踪、切换、归档学习目标
- 📚 **课程管理** - 开始学习、完成课程、重置进度
- 📊 **进度追踪** - 实时查看学习进度和状态
- 🔖 **书签系统** - 记录学习中的疑问点，探索后返回主线
- 💾 **知识缓存** - 本地缓存官方文档和参考资料
- 🔄 **多目标支持** - 同时管理多个学习目标

## 📦 安装

### 方式 1：通过 ClawHub 安装

```bash
clawhub install learn-system-openclaw
```

### 方式 2：手动安装

1. 将 `learn-system-openclaw` 目录复制到你的 OpenClaw skills 目录
2. 确保 `SKILL.md`、`VERSION` 和 `metadata.json` 文件完整

## 🚀 快速开始

### 建议：创建专门的学习 Agent

⚠️ **重要提示**：本技能使用语义匹配来识别学习相关的查询。为了避免与其他技能发生冲突，**强烈建议**创建一个专门用于学习的 Agent：

```bash
# 创建专门的学习 Agent
openclaw agent create learning-agent --skills learn-system-openclaw

# 使用这个 Agent 处理所有学习相关的任务
openclaw session learning-agent
```

### 可能的语义冲突

由于本技能使用语义匹配，以下类型的查询可能会触发它：
- "学习这个 bug"（可能误触发学习系统）
- "学习这个文档"（如果只是想查看文档，不是学习）
- "我的学习计划"（如果只是记录，不是使用学习系统）
- "查看进度"（可能与其他技能的进度查看功能冲突）

**解决方案**：
- 使用专门的学习 Agent（推荐）
- 使用更具体的关键词（例如"查看学习状态"而不是"查看进度"）
- 遇到误触发时，请反馈以帮助改进

## 📖 使用指南

### 创建学习目标

```
用户: 我要学 OpenClaw
系统: 检测到"我要学"，OpenClaw 不在现有课程列表中
      → 引导创建学习目标
      → 询问学习目的、知识水平、期望时长等
      → 生成个性化课程结构
```

### 开始学习课程

```
用户: 开始学习 ai-tools-fundamentals
系统: 显示学习模式选择
      → 快速模式（约 3 天）- 理论为主
      → 完整模式（约 1 周）- 理论+实践
      → 初始化学习状态
```

### 查看学习状态

```
用户: 查看学习状态
系统: 显示所有课程进度概览
      → 总体进度
      → 各优先级进度
      → 每个课程的详细状态
```

### 创建书签

```
用户: 创建书签 Tool Use 机制
系统: 自动获取当前学习位置
      → 记录疑问点和探索目标
      → 保存到书签列表
      → 可随时返回探索
```

## 📂 目录结构

本技能使用 `.learn-system/` 作为根目录，避免与其他技能冲突：

```
.learn-system/
├── goals/                  # 学习目标
│   ├── active/            # 激活的目标
│   ├── paused/            # 暂停的目标
│   ├── completed/         # 已完成的目标
│   └── archived/          # 已归档的目标
├── progress/               # 目标进度追踪
├── bookmarks/             # 书签
├── cache/                  # 知识缓存
├── courses/                # 课程
│   ├── 01-基础入门/
│   ├── 02-进阶探索/
│   └── 03-实战应用/
└── active-goal.md         # 当前激活目标指针
```

## 🎯 核心功能

### 1. 学习目标创建

AI 引导创建个性化学习目标，支持三种类型：
- 📖 **理论型** - "学会使用 X"，按知识主题划分
- 🛠️ **实践型** - "搭建 X"，按项目阶段划分
- 🔄 **混合型** - 理论+实践，分阶段学习

### 2. 课程管理

- **开始学习**：选择学习模式（快速/完整）
- **完成学习**：验证完成度并标记
- **重置课程**：清除所有进度重新开始

### 3. 进度追踪

- **查看总体进度**：所有课程的平均完成度
- **查看课程进度**：特定课程的详细状态
- **更新进度**：解析 checklist.md 自动计算

### 4. 书签系统

- **创建书签**：记录学习中的疑问点
- **继续书签**：返回之前的书签探索
- **完成书签**：探索完成后返回主线学习

### 5. 知识缓存

- **初始化缓存**：从官方文档拉取资料
- **刷新缓存**：更新到最新版本
- **查看缓存**：检查缓存状态

## 🔧 配置

### 自定义根目录

如果需要使用自定义根目录，编辑 `metadata.json`：

```json
{
  "config": {
    "root_dir": ".my-learnings",
    "enable_namespace": true
  }
}
```

### 课程路径映射

在子模块 `status.md` 和 `module-manager.md` 中配置课程路径映射。
