# 学习目标创建 (Goal Creator)

> **精确关键词匹配**：当用户输入包含以下关键词之一时触发此模块

## 触发关键词

- 创建学习目标 / 新建学习目标
- 创建目标 / 新建目标 / 添加目标
- 新学习目标

## 功能

AI 引导用户创建个性化学习目标并生成课程结构。

---

## 目标类型识别

### 类型 1：理论型（📖）

**特征**：用户表达为"学会使用 X"、"了解 X"

**课程结构**：按知识主题划分

### 类型 2：实践型（🛠️）

**特征**：用户表达为"搭建 X"、"构建 X"、"实现 X"

**课程结构**：按项目阶段划分

### 类型 3：混合型（🔄）

**特征**：同时包含学习和实践需求

**课程结构**：理论部分 + 实践部分

## 知识源分析

### 信息收集

1. **搜索官方文档**
   - 查找项目官网
   - 识别官方文档链接
   - 获取 GitHub 仓库

2. **识别核心概念**
   - 从文档中提取关键术语
   - 识别主要功能模块
   - 分析技能依赖关系

3. **分析难度等级**
   - 入门门槛（前置知识要求）
   - 学习曲线（概念复杂度）
   - 实践难度（环境搭建、依赖关系）

## 课程生成算法

### 理论型课程生成

```
输入：知识源分析结果
输出：课程大纲（按主题）

算法：
1. 从核心概念列表中选择主要概念
2. 按"基础 → 进阶 → 高级"排序
3. 每个概念成为一个模块
4. 每个模块包含：
   - 概念定义
   - 使用场景
   - 相关命令/配置
   - 实践示例
```

### 实践型课程生成

```
输入：项目需求分析
输出：课程大纲（按阶段）

算法：
1. 分析项目开发流程
2. 按阶段划分：规划 → 搭建 → 实现 → 测试 → 部署
3. 每个阶段成为一个模块
4. 每个模块包含：
   - 阶段目标
   - 所需技能
   - 具体任务
   - 验收标准
```


## 注意事项

1. **交互式询问**：逐步询问，一次一个问题
2. **用户确认**：创建课程前必须显示预览并等待确认
3. **知识源分析**：使用 web-search 或 webReader 获取信息
4. **个性化适配**：根据用户水平调整内容深度
5. **逐模块确认**：允许用户调整模块内容和顺序
