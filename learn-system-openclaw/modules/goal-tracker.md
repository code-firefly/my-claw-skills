# 学习目标追踪 (Goal Tracker)

> **精确关键词匹配**：当用户输入包含以下关键词之一时触发此模块

## 触发关键词

- 查看学习目标 / 显示学习目标 / 学习目标列表
- 我的学习目标 / 所有学习目标
- 学习目标概览 / 学习目标 / 看看学习目标
- 当前学习目标
- 学习目标状态
- 切换学习目标
- 暂停学习目标 / 恢复学习目标
- 归档学习目标
- 完成目标

## 功能

管理多个学习目标的生命周期和状态切换。

---

## 目标状态

| 状态 | 说明 | 目录 |
|------|------|------|
| **active** | 当前进行中的目标 | `.learn-system/goals/active/` |
| **paused** | 暂停的目标 | `.learn-system/goals/paused/` |
| **completed** | 已完成的目标 | `.learn-system/goals/completed/` |
| **archived** | 已归档的目标（软删除） | `.learn-system/goals/archived/` |

## 切换学习目标

**执行步骤**：

### 步骤 1：检查目标状态

从目标列表中查找目标及其当前状态

### 步骤 2：处理不同状态

| 当前状态 | 操作 |
|----------|------|
| **active** | 提示"该目标已是当前目标" |
| **paused** | 询问确认后激活 |
| **completed** | 询问"是否重新开始？" |
| **archived** | 询问"是否从归档恢复？" |

### 步骤 3：保存当前目标进度

如果当前有激活的目标，先保存其进度到对应的 progress 文件

### 步骤 4：加载新目标

1. 读取新目标的进度文件
2. 读取新目标的书签和笔记
3. 更新 `active-goal.md` 引用

### 步骤 5：显示切换结果

```
✅ 目标已切换！

🎯 新目标：learn-openclaw
📊 进度：35% ███████░░░░░░░░░░░░
📅 下一步：完成 MCP 协议基础学习

当前课程模块：
• MCP 协议入门 (进行中 60%)
• AI 编排平台 (未开始)
```

## 暂停学习目标

**执行步骤**：

1. 保存当前进度
2. 将目标文件从 `active/` 移动到 `paused/`
3. 更新 `active-goal.md` 为无目标状态
4. 显示暂停信息

---

在目标文件中添加完成日期和总学习时长

### 步骤 4：清除当前目标

更新 `active-goal.md` 为无目标状态

## 归档学习目标

**执行步骤**：

1. 显示警告：归档后目标数据永久保留但不再显示在主列表
2. 询问确认（需要输入目标名）
3. 将目标文件从当前状态目录移动到 `archived/`
4. 显示归档完成信息

**注意**：归档是软删除，数据永久保留

## 注意事项

1. **状态转换规则**：
   - active → paused（暂停）
   - paused → active（恢复）
   - active → completed（完成）
   - 任何状态 → archived（归档）
   - archived → active（恢复）

2. **进度保存**：切换目标前必须保存当前目标的进度

3. **数据验证**：移动目标文件后验证数据完整性
