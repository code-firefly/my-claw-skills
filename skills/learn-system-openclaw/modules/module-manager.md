## 前置步骤（所有课程操作前必须执行）

```bash
# 步骤 1：确保目录存在
exec: mkdir -p .learn-system

# 步骤 2：读取进度文件

```

**禁止**：跳过以上任何步骤。

## 开始学习

### 执行步骤

**步骤 1**：读取 `.learn-system/progress.json`

**步骤 2**：检查课程是否存在

从 `.learn-system/cache/.metadata.json` 读取缓存列表。

**步骤 3**：验证课程状态

检查课程是否已开始。

**步骤 4**：选择学习模式

询问用户：
- 1. 📚 快速模式（约 3 天）- 理论为主
- 2. 🛠️ 完整模式（约 1 周）- 理论+实践

**模式说明**：
- 快速模式：适合快速了解概念，以阅读文档为主
- 完整模式：深入掌握技能，包含实践练习

**步骤 5**：显示更新预览

```

║                  开始学习预览                         ║
╠═════════════════════════════════════════════════════╣
║ 课程：ai-tools-fundamentals                           ║
║ 优先级：P0 🔴 必学                                     ║
║ 模式：📚 快速模式（约 3 天）                           ║
║ 开始日期：2026-03-13                                   ║
╠═════════════════════════════════════════════════════╣
║ 将要更新：                                             ║
║   ✓ .learn-system/progress.json                           ║
╠═════════════════════════════════════════════════════╣
║ 确认开始学习？(y/n)                                    ║
╚═════════════════════════════════════════════════════╝
```

**步骤 6**：用户确认后，更新 `.learn-system/progress.json`

## 重置课程

### 执行步骤

**步骤 1**：读取 `.learn-system/progress.json`

**步骤 2**：显示重置警告

```
⚠️  此操作将清除课程的所有进度

课程：<course-name>
重置后将重新开始，确认继续？(yes/no)
```

**步骤 3**：用户确认后，重置进度

将所有模块状态重置为 `not_started`，`overall_progress` 设为 0。

## ⚠️ 注意事项

### 通用注意事项

1. **路径强制**：必须使用 `.learn-system/progress.json`，不得使用其他路径
2. **用户确认**：修改多个文件前，总是先显示预览并等待用户确认
3. **日期格式**：统一使用 YYYY-MM-DD 格式
4. **文件路径**：所有路径使用相对路径（`.learn-system/`），避免绝对路径或 `~`

### JSON 操作注意事项

1. **读取 JSON**：使用 `read` 工具读取，然后解析 JSON
2. **修改 JSON**：解析后修改 JSON 对象，然后写回
3. **写回 JSON**：使用 `write` 工具完整替换文件
4. **错误处理**：如果 JSON 解析失败，报告明确错误并停止
