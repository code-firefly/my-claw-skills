# 学习状态管理 (Status)

> **精确关键词匹配**：当用户输入包含以下关键词之一时触发此模块

## 🚨 强制性路径规则 ⚠️

**必须**使用 `.learn-system/` 作为根目录。

**禁止**：
- 使用任何其他路径
- 让模型自主决定文件位置

---

## 触发关键词

- 查看学习状态 / 学习状态
- 更新进度 / 进度更新
- 查看进度 / 显示进度 / 课程进度
- 打勾

---

## 功能

提供状态管理和进度追踪功能。

---

## 前置步骤（所有操作前必须执行）

```bash
# 步骤 1：确保目录存在
exec: mkdir -p .learn-system

# 步骤 2：读取现有数据（如果文件存在）
read: .learn-system/progress.json
```

**禁止**：跳过以上任何步骤。

## 更新进度

### 执行步骤

**步骤 1**：读取 `.learn-system/progress.json`

**步骤 2**：解析用户输入，提取模块名称和完成状态

**步骤 3**：更新模块状态

**步骤 4**：重新计算总体进度

**步骤 5**：写回 `.learn-system/progress.json`

**步骤 6**：显示更新结果

```
✅ 进度已更新

课程：<course-name>
模块：<module-name>
新状态：已完成
总体进度：XX%
```

## ⚠️ 注意事项

### 通用注意事项

1. **路径强制**：必须使用 `.learn-system/` 前缀
2. **JSON 操作**：读取、解析、修改、写回 JSON 文件
3. **错误处理**：如果 JSON 解析失败，报告明确错误
4. **日期格式**：统一使用 YYYY-MM-DD 格式

### 路径注意事项

**必须遵守**：
- ✅ 书签文件：`.learn-system/bookmarks.json`
- ✅ 进度文件：`.learn-system/progress.json`
- ✅ 目标文件：`.learn-system/goals.json`
- ✅ 缓存目录：`.learn-system/cache/`
- ❌ 禁止：使用任何其他路径

### JSON 操作注意事项

1. **读取 JSON**：使用 `read` 工具读取，然后解析
2. **修改 JSON**：解析后修改对象，然后写回
3. **写回 JSON**：使用 `write` 工具完整替换文件
4. **错误处理**：如果 JSON 解析失败，报告错误并停止
